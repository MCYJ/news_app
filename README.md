# Project: News Now
## 각국의 최신 뉴스 및 커뮤니티 서비스를 제공하는 어플리케이션 프로젝트

### 기능구성
1. 로그인 및 회원가입
  - 각 페이지 UI 구성 및 회원가입 정보 검증(validation)
2. 실시간 뉴스 업데이트
  - Retrofit을 활용한 Restful API 요청 전송 및 응답처리
3. 뉴스 관련 커뮤니티 이용
